# picodraw
a smol gpu graphics library

## Installation

To install `picodraw` add this to your `Cargo.toml`:

```toml
[dependencies]
picodraw = { git = "https://github.com/blepfx/picodraw", features = ["derive", "opengl"], branch = "rewrite" }
```
